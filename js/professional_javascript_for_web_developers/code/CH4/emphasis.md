分内给Web浏览器的可用内存数量通常要比分配给桌面的应用程序少。


确保占用最少的内存可以让页面获得更好的性能， 而优化内存占用的最佳方式，就是未执行中的代码只保存必要的数据。

一旦数据不再有用最好通过将其设置为null来释放起引用。
